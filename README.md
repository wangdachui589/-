# -
vue作业  在线论坛 

（1）页面制作美观大方、功能完善，代码书写规范，变量命名符合软件开发通用标准；使用Vue脚手架开发，文件命名规范；页面可以在不同类型的浏览器中正常显示，能够体现一定的跨平台脚本开发水平。
（2）页面底部添加个人信息（姓名、班级、学号）并动态显示当前时间，要求：时间精确到秒，格式：x月x日xx:xx:xx；时间能够每秒自动刷新一次。
（3）页面中设计有图片轮播区域，能够实现无限轮播功能，并提供控制轮播方向按钮（样式自定）。要求：页面打开后，能够实现图片自动循环轮播，第一幅图片和最后一幅图片可以实现无缝衔接，没有空白或其他不符合轮播动画要求的现象发生；轮播方向按钮能够控制图片运动方向。
（4）页面中实现“论坛导航”动画二级菜单功能（位置自定）。要求：单击一级分类菜单后，实现二级菜单下滑出现、上滑隐藏的动画效果；
（5）制作帖子展示区，能够显示帖子信息介绍（至少包含：帖子名称、图片、详细内容等信息），能够实现点赞、评论功能。要求：能够判断当前帖子是否点赞，重复点击则实现点赞和取消功能。
